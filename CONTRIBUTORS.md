# Contributors

AIACTA is an open-source project built by a global community of developers, creators, and policy experts. We are grateful for every contribution that helps build a fair AI attribution ecosystem.

*Founding Author: Eric Michel, PhD*

---

## How to get listed
We follow the [All Contributors](https://allcontributors.org) specification. To add yourself to this list, please open a Pull Request or comment on an issue you've resolved with:
`@aiacta-bot please add @username for code, docs, tests`
